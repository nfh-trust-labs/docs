# Registry Creation and Management Guide

This document explains the complete flow for creating registries and managing **dedi.global**.

### 🪪 Account Setup

1. **Create an Account**\
   Sign up for an account on the platform.
2. **Verify Your Account**\
   Check your email and click on the verification link you receive to activate your account.
3. **Generate an API Key (Optional)**\
   Once your account is verified, you can generate an API key for secure programmatic access.

### 🧩 Namespace Setup

1. **Create a Namespace**\
   Use the dashboard or API to create a new namespace.
2. **Verify Your Namespace (Optional)**\
   Follow the instructions in domain-verification to verify your namespace.

### 🗂️ Registry and Records

1. **Create a Registry**\
   After setting up your namespace, create a new registry under it.
2. **Add Delegates (Optional)**\
   Assign delegates to help manage your registry.
3. **Add Records**\
   Start adding records to your registry once setup is complete.

### 🔄 Updating Information

After setup, you can update:

* Namespace details
* Registry details
* Record details

as needed through the dashboard or APIs.

### ⚙️ State Management

Different state management options are available for registries and records.\
👉 Explore state management.

### 🌐 Accessing Data

Use the Standard APIs to query and manage data.\
For usage patterns and sample implementations, refer to the `examples` directory.
