# Developer documentation

This contains the documentation for the Dedi global. The docs are organized into topic files (Overview, Standard APIs, Guides, Quickstart, Resources). Each folder contains one or more  files that explain concepts, show examples, and provide step-by-step instructions.

### Table of contents

1. [DeDi 101 ](dedi-101.md)- The Overview section introduces Dedi and describes the architecture.
2. [Standard APIs](standard-apis/) - This section documents the standard APIs defined by the Dedi protocol. It explains lookup, query, versioning, and other core API behaviors
3. [Guides](guides/) - Practical how-tos and walkthroughs are in the Guides folder (`3. guides`). Topics include domain verification, schema design, bulk upload, delegation, and state management.
4. [Quickstart](quickstart/) - The Quickstart explains the available environments and the exact flow to set up a local registry and try Dedi.
5. [Resources](/broken/pages/uAIlX2nOgNZalWtLXAyF) - Additional reference material (FAQ, glossary, roadmap)
